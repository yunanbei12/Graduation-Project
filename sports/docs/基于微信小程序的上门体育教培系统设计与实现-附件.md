### 2.3 课程下单与课包流转流程图

```mermaid
flowchart TD
    A["用户浏览课程"] --> B["进入课程详情"]
    B --> C["确认课程订单"]
    C --> D["创建订单"]
    D --> E{"是否支付成功"}
    E -- 否 --> F["延迟关闭订单"]
    E -- 是 --> G["生成私教课包或团课报名记录"]
    G --> H["课程进行中"]
    H --> I["销课 / 上课"]
    I --> J{"课时是否完成"}
    J -- 否 --> H
    J -- 是 --> K["课包状态=已完成"]
    K --> L["订单状态=已完成"]
```

### 2.4 商品下单与收货流程图

```mermaid
flowchart TD
    A["浏览商品"] --> B["商品详情"]
    B --> C["加入购物车或立即购买"]
    C --> D["确认订单并选择地址"]
    D --> E["创建商品订单"]
    E --> F{"支付是否成功"}
    F -- 否 --> G["订单超时自动取消"]
    F -- 是 --> H["库存扣减"]
    H --> I["后台发货"]
    I --> J["用户确认收货"]
    J --> K["订单完成"]
```

### 2.5 AI 客服处理流程图

```mermaid
flowchart TD
    A["用户发送咨询"] --> B["创建或续用会话"]
    B --> C["识别咨询意图"]
    C --> D["查询订单/课包/优惠券等业务数据"]
    D --> E["生成 AI 回复"]
    E --> F{"用户是否满意"}
    F -- 是 --> G["结束本轮咨询"]
    F -- 否 --> H{"是否转人工"}
    H -- 否 --> A
    H -- 是 --> I["创建转人工记录"]
    I --> J["后台客服接入"]
    J --> K["人工回复并标记状态"]
```
