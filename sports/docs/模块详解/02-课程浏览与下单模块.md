# 课程浏览与下单模块详解

> 本模块是系统最核心的业务主链路：课程列表 → 课程详情 → 确认订单 → 创建订单 → 支付
> 覆盖私教课包和团课两种课程形态，链路略有差异。

---

## 一、模块全链路概览

```
课程列表页（course.vue）
  ├── 私教课 Tab → 私教课详情（course-detail-private.vue）→ 确认订单（confirm-private.vue）
  └── 团课 Tab   → 团课详情（course-detail-group.vue）  → 选择排课 → 确认订单（confirm-group.vue）
                                                                          ↓
                                                              POST /order/course（创建订单）
                                                                          ↓
                                                              OrderServiceImpl.createCourseOrder()
                                                                          ↓
                                                              校验课程/排课/手机号/优惠券
                                                              写入 order + order_item 表
                                                              锁定优惠券
                                                              发布延迟关单消息（RabbitMQ 15分钟）
                                                                          ↓
                                                              跳转订单详情页 → 用户点击支付
                                                                          ↓
                                                              PUT /order/pay/{id}
                                                                          ↓
                                                              OrderServiceImpl.doPay()
```

```javascript
async loadCourses() {
  const type = this.activeTab === 'private' ? 1 : 2  // 1=私教 2=团课
  const params = { pageNum: 1, pageSize: 20, type }
  if (this.categoryId) params.categoryId = this.categoryId

  const res = await getCourseList(params)  // GET /course/list
  const records = res.records || []

  if (type === 1) {
    // 私教课：直接渲染卡片
    this.privatePackages = records.map(c => ({ id: c.id, name: c.name, price: c.price, ... }))
  } else {
    // 团课：先渲染基础信息，再异步加载每个课程的最近排课
    this.groupClasses = records.map(c => ({ id: c.id, name: c.name, time: '', enrolled: 0, ... }))
    groupItems.forEach((item, idx) => {
      this.loadScheduleForItem(item.id, idx)  // GET /course/schedule/list?courseId=xxx
    })

```

**团课的两步加载**是一个设计亮点：先快速渲染课程基础信息，再异步补充排课时间和名额，避免用户等待。

```java
// 课程列表（公开接口，不需要登录）
@GetMapping("/list")
public ServerResponseEntity<Page<Course>> list(
        @RequestParam(required = false) Integer type,       // 1=私教 2=团课
        @RequestParam(required = false) Long categoryId) {
    LambdaQueryWrapper<Course> wrapper = new LambdaQueryWrapper<>();
    wrapper.eq(Course::getStatus, 1);                       // 只查上架课程
    if (type != null) wrapper.eq(Course::getType, type);
    if (categoryId != null) wrapper.eq(Course::getCategoryId, categoryId);
    wrapper.orderByDesc(Course::getSales);                  // 按销量降序
    return ServerResponseEntity.success(courseService.page(page, wrapper));
}

// 团课排课列表（公开接口）
@GetMapping("/schedule/list")
public ServerResponseEntity<Page<CourseSchedule>> scheduleList(@RequestParam Long courseId) {
    return ServerResponseEntity.success(courseScheduleService.page(page,
        new LambdaQueryWrapper<CourseSchedule>()
            .eq(CourseSchedule::getCourseId, courseId)
            .in(CourseSchedule::getStatus, 0, 1)  // 0=未开始可预约 1=进行中
            .gt(CourseSchedule::getTotalSeats, 0)
            .orderByAsc(CourseSchedule::getScheduleDate)));
}
```

> 课程列表和排课列表都是**公开接口**，在 Sa-Token 白名单中，不需要登录即可访问。

```javascript
pickCoupon(uc) {
  this.selectedCoupon = uc
  const price = parseFloat(this.course.price)
  if (uc.couponType === 1 || uc.couponType === 3) {
    this.couponAmount = parseFloat(uc.couponDiscount)         // 满减/无门槛：固定金额
  } else if (uc.couponType === 2) {
    this.couponAmount = price - price * parseFloat(uc.couponDiscountRatio)  // 折扣券
  }
  this.couponAmount = Math.min(this.couponAmount, price)  // 不超过原价
}
```

**实付金额计算**（前端实时展示，最终以后端为准）：

```javascript
computed: {
  actualAmount() {
    const price = parseFloat(this.course.price) || 0
    return Math.max(0, price - this.couponAmount).toFixed(2)
  }
}
```

**提交订单**：

```javascript
async onPay() {
  // 1. 手机号绑定校验（读本地缓存，快速拦截）
  const userInfo = JSON.parse(uni.getStorageSync('userInfo') || '{}')
  if (!userInfo.phone) {
    uni.showModal({ title: '提示', content: '下单前需绑定手机号', ... })

}
```

## 六、后端创建订单（核心）

**文件路径**：`OrderServiceImpl.java`

### 6.1 创建课程订单完整逻辑

```java
@Transactional(rollbackFor = Exception.class)
public Order createCourseOrder(Long userId, Order params) {

    // ① 用户校验
    User currentUser = requireUser(userId);
    ensurePhoneBound(currentUser);  // 必须绑定手机号，否则抛异常

    // ② 课程校验
    Course course = courseService.getById(params.getCourseId());
    if (course == null || course.getStatus() != 1) {
        throw new IllegalArgumentException("课程不存在或已下架");
    }

    // ③ 团课额外校验（私教课跳过此块）
    if (course.getType() == 2) {
        CourseSchedule schedule = courseScheduleService.getById(params.getScheduleId());
        if (schedule == null || schedule.getStatus() != 0) {
            throw new IllegalArgumentException("所选排课不存在或已无法报名");
        }
        // 时间校验：不能报已开始的场次
        LocalDateTime scheduleStart = resolveStartTime(schedule, course);
        if (scheduleStart != null && !scheduleStart.isAfter(LocalDateTime.now())) {
            throw new IllegalArgumentException("该场次已开始，无法报名");
        }
        // 名额校验
        if (schedule.getEnrolledSeats() >= schedule.getTotalSeats()) {
            throw new IllegalArgumentException("该排课名额已满");
        }
        // 重复报名校验
        long existCount = this.count(new LambdaQueryWrapper<Order>()
            .eq(Order::getUserId, userId)
            .eq(Order::getScheduleId, params.getScheduleId())
            .in(Order::getStatus, 1, 2, 3, 4, 6));  // 待支付/已支付/处理中/完成/退款中
        if (existCount > 0) {
            throw new IllegalArgumentException("您已报名该场次，请勿重复购买");
        }
        order.setScheduleId(params.getScheduleId());
    }

    // ④ 优惠券计算（校验 + 计算折扣金额）
    CouponPricing pricing = calculateCouponPricing(userId, params.getCouponId(), totalAmount, 1);
    order.setTotalAmount(totalAmount);
    order.setCouponAmount(pricing.couponAmount());
    order.setActualAmount(totalAmount.subtract(pricing.couponAmount()));

    // ⑤ 写入订单和订单项（同一事务）
    this.save(order);
    orderItemService.save(item);

    // ⑥ 锁定优惠券（status 改为 3=LOCKED，防止重复使用）
    lockCouponForOrder(pricing.userCouponId(), userId, order.getId());

    // ⑦ 事务提交后发布延迟关单消息（15分钟后自动取消）
    txEventHelper.afterCommit(() ->
        orderEventPublisher.publishOrderCloseDelay(order.getId())
    );

    return order;
}
```

### 6.2 订单状态码

| 状态码 | 含义 |
|--------|------|
| 1 | 待支付 |
| 2 | 已支付 |
| 3 | 待处理/履约中 |
| 4 | 已完成 |
| 5 | 已取消 |
| 6 | 退款中 |
| 7 | 已退款 |

### 6.3 优惠券计算逻辑

```java
private CouponPricing calculateCouponPricing(Long userId, Long userCouponId,
                                              BigDecimal totalAmount, int orderType) {
    if (userCouponId == null) return new CouponPricing(null, BigDecimal.ZERO);

    UserCoupon userCoupon = userCouponService.getById(userCouponId);
    // 校验：属于该用户 + 未使用状态
    if (userCoupon == null || !userCoupon.getUserId().equals(userId)
        || userCoupon.getStatus() != 0) {
        throw new IllegalArgumentException("优惠券不可用");
    }

    Coupon coupon = couponService.getById(userCoupon.getCouponId());
    // 校验使用门槛
    if (totalAmount.compareTo(coupon.getMinAmount()) < 0) {
        throw new IllegalArgumentException("订单金额未达到优惠券使用门槛");
    }

    BigDecimal couponAmount;
    if (coupon.getType() == 1) {
        couponAmount = coupon.getDiscount();                                    // 满减券
    } else if (coupon.getType() == 2) {
        couponAmount = totalAmount.subtract(totalAmount.multiply(coupon.getDiscountRatio()));  // 折扣券
    } else {
        couponAmount = coupon.getDiscount();                                    // 无门槛券
    }
    couponAmount = couponAmount.min(totalAmount);  // 不超过订单金额
    return new CouponPricing(userCouponId, couponAmount);
}
```

### 7.2 后端支付处理（doPay）

```java
private void doPay(Order order) {
    // 分布式锁：防止并发重复支付
    RLock orderLock = redissonClient.getLock("order:pay:" + order.getId());
    orderLock.lock();
    try {
        if (order.getStatus() != ORDER_PENDING) {
            throw new IllegalArgumentException("订单已支付或已关闭");
        }
        order.setStatus(ORDER_PAID);
        order.setPayTime(LocalDateTime.now());
        this.updateById(order);

        if (order.getOrderType() == 1) {
            handleCourseOrderPaid(order);   // 课程订单：生成课包 或 占用团课名额
        } else {
            handleProdOrderPaid(order);     // 商品订单：扣减 SKU 库存
        }

        consumeCoupon(order.getCouponId(), order.getUserId());  // 核销优惠券（status=1）
        updateSales(order);                                     // 更新销量

        // 事务提交后异步发布支付成功事件（触发满额发券等）
        txEventHelper.afterCommit(() ->
            orderEventPublisher.publishOrderPaid(new OrderLifecycleMessage(order))
        );
    } finally {
        orderLock.unlock();
    }
}
```

### 7.4 团课支付后：占用名额 + 触发成团检查

```java
// 分布式锁防止并发超卖
RLock scheduleLock = redissonClient.getLock("order:schedule:" + order.getScheduleId());
scheduleLock.lock();
try {
    CourseSchedule schedule = courseScheduleService.getById(order.getScheduleId());
    if (schedule.getEnrolledSeats() >= schedule.getTotalSeats()) {
        throw new IllegalArgumentException("名额已满，支付失败");
    }
    schedule.setEnrolledSeats(schedule.getEnrolledSeats() + 1);  // 占用名额
    courseScheduleService.updateById(schedule);

    // 触发成团检查（发布延迟消息，开课前检查是否达到最低成团人数）
    scheduleGroupCheck(order.getScheduleId(), schedule, course);
} finally {
    scheduleLock.unlock();
}
```

### 7.5 延迟关单机制（RabbitMQ）

```java
// 下单后（事务提交后）发布延迟消息
txEventHelper.afterCommit(() ->
    orderEventPublisher.publishOrderCloseDelay(order.getId())
);

// RabbitMQ 延迟队列：TTL = 15分钟，超时后消息转入死信队列
@Bean
public Queue orderCloseDelayQueue(OrderBizProperties properties) {
    return QueueBuilder.durable(ORDER_CLOSE_DELAY_QUEUE)
        .ttl(properties.getCloseTimeoutMinutes() * 60 * 1000)  // 15分钟
        .deadLetterExchange(ORDER_EVENT_EXCHANGE)
        .deadLetterRoutingKey(ORDER_CLOSE_PROCESS_ROUTING_KEY)
        .build();
}

// 消费死信队列，执行关单
@RabbitListener(queues = RabbitConfig.ORDER_CLOSE_PROCESS_QUEUE)
public void handleOrderClose(OrderDelayMessage message) {
    orderService.autoCancelOrder(message.getOrderId(), "超时未支付，系统自动取消");
}
```

## 十、关键设计点总结

| 设计点 | 实现方式 | 目的 |
|--------|----------|------|
| 团课两步加载 | 先渲染基础信息，再异步补充排课 | 提升页面响应速度 |
| 前端手机号校验 | 下单前读取本地缓存的 userInfo.phone | 快速拦截，减少无效请求 |
| 后端手机号校验 | ensurePhoneBound() | 兜底保障 |
