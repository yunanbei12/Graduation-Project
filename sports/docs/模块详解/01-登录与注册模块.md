# 登录与注册模块详解

## 一、模块概览

本系统有两套独立的登录体系：

| 登录体系 | 使用者 | 入口 | 后端接口 |
|----------|--------|------|----------|
| 小程序用户登录 | 学员用户 | 小程序登录页 | 用户端接口 8086 |
| 管理后台登录 | 运营/教务/客服等管理员 | 浏览器管理后台 | 后台管理接口 8085 |

两套体系共用同一套 Sa-Token 鉴权框架，但账号体系完全独立：
- 小程序用户存储在 `user` 表
- 管理员存储在 `sys_user` 表

### 2.3 手机号密码登录流程

```
用户输入手机号+密码
  → 前端表单校验（validate）
  → 调用 accountLogin(phone, passWord)
  → POST http://localhost:8086/auth/login
  → 后端查库校验密码（BCrypt）
  → 返回 { token, userId, nickName, avatarUrl, phone }
  → 存入 uni.setStorageSync('token') 和 ('userInfo')
  → 跳转回原页面
```

**前端调用**（`accountLogin.vue`）：

```javascript
  this.loading = true
  try {
    const res = await accountLogin(this.form.phone.trim(), this.form.passWord)
    this.onLoginSuccess(res)
  } finally {
    this.loading = false
  }
}
```

**API 封装**（`api/auth.js`）：

**HTTP 工具层**（`utils/http.js`）：

```javascript
function request(options) {
  return new Promise((resolve, reject) => {
    uni.request({
      url: BASE_URL + options.url,  // http://localhost:8086/auth/login
      method: options.method,
      data: options.data,
      header: {
        'Content-Type': 'application/json',
        'Authorization': getToken()  // 登录时为空字符串
      },
      success: (res) => {
        if (res.data.code === 200) {
          resolve(res.data.data)   // 只返回 data 字段给业务层
        } else if (res.data.code === 401) {
          uni.removeStorageSync('token')
          uni.reLaunch({ url: '/pages/profile/profile' })
          reject(res.data)
        } else {
          reject(res.data)
        }
      },
      fail: (err) => {
        uni.showToast({ title: '网络连接失败', icon: 'none' })
        reject(err)
      }
    })
  })
}
```

**后端接口**（`ApiAuthController.java`，端口 8086）：

```java
@PostMapping("/login")
public ServerResponseEntity<Map<String, Object>> login(@RequestBody Map<String, String> params) {
    String phone = params.get("phone");
    String passWord = params.get("passWord");

    // 1. 根据手机号查找用户
    User user = userService.getOne(
        new LambdaQueryWrapper<User>().eq(User::getPhone, phone)
    );
    if (user == null) {
        return ServerResponseEntity.fail("手机号或密码不正确");
    }

    // 2. 未设置密码的用户（验证码注册）给出友好提示
    if (user.getLoginPassword() == null || user.getLoginPassword().isEmpty()) {
        return ServerResponseEntity.fail("您尚未设置登录密码，请使用验证码登录");
    }

    // 3. BCrypt 密码比对
    if (!passwordEncoder.matches(passWord, user.getLoginPassword())) {
        return ServerResponseEntity.fail("手机号或密码不正确");
    }

    // 4. 账号状态检查
    if (user.getStatus() != 1) {
        return ServerResponseEntity.fail("账号已被禁用");
    }

    // 5. Sa-Token 登录，生成 token
    StpUtil.login(user.getId());
    String token = StpUtil.getTokenValue();

    Map<String, Object> result = new HashMap<>();
    result.put("token", token);
    result.put("userId", user.getId());
    result.put("nickName", user.getNickName());
    result.put("avatarUrl", user.getAvatarUrl());
    result.put("phone", user.getPhone());
    return ServerResponseEntity.success(result);
}
```

**统一响应体格式**（`ServerResponseEntity.java`）：

```json
  "data": {
    "token": "xxxx-xxxx-xxxx",
```

### 2.4 短信验证码登录流程

```
输入手机号 → 点击"获取验证码"
  → POST /sms/send { phone, type: "0" }
  → 后端发送短信（开发环境输出到控制台日志）
  → 前端启动 60 秒倒计时

输入验证码 → 点击"立即登录"
  → POST /auth/smsLogin { phone, code }
  → 后端校验验证码
    ├── 已注册 → 直接登录，返回 token
    └── 未注册 → 返回 { notRegistered: true }
              → 前端弹窗"是否注册并登录？"
              → 用户确认 → POST /auth/smsLogin { phone, code, confirmRegister: "true" }
              → 后端自动创建账号 → 登录 → 返回 token
```

**后端短信登录接口**（`ApiAuthController.java`）：

```java
@PostMapping("/smsLogin")
public ServerResponseEntity<Map<String, Object>> smsLogin(@RequestBody Map<String, String> params) {
    String phone = params.get("phone");
    String code = params.get("code");
    String confirmRegister = params.get("confirmRegister");

    // 1. 手机号格式校验
    if (!PHONE_PATTERN.matcher(phone).matches()) {
        return ServerResponseEntity.fail("请输入正确的手机号");
    }

    // 2. 验证码校验（confirmRegister=true 时跳过，上一步已消耗）
    if (!"true".equals(confirmRegister)) {
        if (!smsLogService.verifyCode(phone, code.trim(), 0)) {
            return ServerResponseEntity.fail("验证码错误或已过期");
        }
    }

    // 3. 查找用户
    User user = userService.getOne(
        new LambdaQueryWrapper<User>().eq(User::getPhone, phone)
    );

    if (user == null) {
        if (!"true".equals(confirmRegister)) {
            // 未注册且未确认，返回提示让前端弹窗
            Map<String, Object> result = new HashMap<>();
            result.put("notRegistered", true);
            return ServerResponseEntity.success(result);
        }
        // 前端已确认，自动创建账号
        user = new User();
        user.setPhone(phone);
        user.setNickName("用户" + RandomUtil.randomNumbers(4));
        user.setRegisterType(2); // 短信验证码注册
        user.setStatus(1);
        userService.save(user);
        sendRegisterGiftCoupons(user.getId()); // 发放注册赠券
    }

    // 4. Sa-Token 登录
    StpUtil.login(user.getId());
    String token = StpUtil.getTokenValue();
    // ... 返回 token 和用户信息
}
```

### 2.5 微信登录流程

```
小程序调用 wx.login() 获取 code
  → POST /auth/wxLogin { code }
  → 后端请求微信服务器换取 openId
  → 根据 openId 查找用户
    ├── 已存在 → 直接登录
    └── 不存在 → 自动注册（registerType=0）→ 发放注册赠券 → 登录
  → 返回 token
```

**后端换取 openId**（`ApiAuthController.java`）：

```java
private String getOpenIdByCode(String code) {
    String url = String.format(
        "https://api.weixin.qq.com/sns/jscode2session?appid=%s&secret=%s&js_code=%s&grant_type=authorization_code",
        wxAppid, wxSecret, code
    );
    String response = HttpUtil.get(url);
    cn.hutool.json.JSONObject json = new cn.hutool.json.JSONUtil().parseObj(response);
    return json.getStr("openid");
}
```

### 2.6 注册赠券机制

每次新用户注册（无论哪种方式），都会自动发放"注册赠送"优惠券：

```java
private void sendRegisterGiftCoupons(Long userId) {
    // 查找所有启用且标记为注册赠送的优惠券
    List<Coupon> giftCoupons = couponService.list(
        new LambdaQueryWrapper<Coupon>()
            .eq(Coupon::getStatus, 1)
            .eq(Coupon::getRegisterGift, 1)
    );
    for (Coupon coupon : giftCoupons) {
        // 防止重复发放
        long count = userCouponService.count(...);
        if (count > 0) continue;
        // 检查发放总量限制
        if (coupon.getTotalCount() > 0 && coupon.getUsedCount() >= coupon.getTotalCount()) continue;
        // 发放
        UserCoupon userCoupon = new UserCoupon();
        userCoupon.setUserId(userId);
        userCoupon.setCouponId(coupon.getId());
        userCoupon.setStatus(0); // 未使用
        userCouponService.save(userCoupon);
        coupon.setUsedCount(coupon.getUsedCount() + 1);
        couponService.updateById(coupon);
    }
}
```

### 2.7 登录守卫（需要登录的页面）

**文件路径**：`utils/auth.js`

部分页面需要登录才能访问，通过 `checkLogin()` 方法统一处理：

```javascript
const AUTH_PAGES = [

export function checkLogin(fromPage) {
  if (isLoggedIn()) return true
  const currentPage = fromPage || getCurrentPagePath()
  uni.navigateTo({
    // 携带 redirect 参数，登录成功后跳回原页面

  })
  return false
}

// 登录成功后跳转回原页面
export function navigateBackAfterLogin(redirectPath) {
  if (redirectPath) {
    const path = decodeURIComponent(redirectPath)
    const tabBarPages = ['pages/index/index', 'pages/course/course', 'pages/mall/mall', 'pages/profile/profile']
    const isTabBar = tabBarPages.some(p => path.startsWith(p))
    if (isTabBar) {
      uni.switchTab({ url: path.split('?')[0] })
    } else {
      uni.redirectTo({ url: path })
    }
  } else {
    uni.switchTab({ url: '/pages/profile/profile' })
  }
}
```

## 三、小程序端注册（全链路）

**文件路径**：`sports-custom-mini/pages/accountLogin/register.vue`

注册需要：手机号 + 短信验证码 + 密码 + 确认密码

**注册流程**：

```
输入手机号+验证码+密码 → 前端校验 → POST /auth/register
  → 后端校验手机号格式
  → 校验验证码
  → 检查手机号是否已注册
  → 密码长度校验（>=6位）
  → BCrypt 加密密码
  → 创建用户记录（registerType=1）
  → 发放注册赠券
  → 返回成功
  → 前端跳转到登录页
```

**后端注册接口**（`ApiAuthController.java`）：

```java
@PostMapping("/register")
public ServerResponseEntity<Map<String, Object>> register(@RequestBody Map<String, String> params) {
    String phone = params.get("phone");
    String passWord = params.get("passWord");
    String code = params.get("code");

    if (!PHONE_PATTERN.matcher(phone).matches()) {
        return ServerResponseEntity.fail("请输入正确的手机号");
    }
    if (!smsLogService.verifyCode(phone, code.trim(), 0)) {
        return ServerResponseEntity.fail("验证码错误或已过期");
    }
    long count = userService.count(
        new LambdaQueryWrapper<User>().eq(User::getPhone, phone)
    );
    if (count > 0) {
        return ServerResponseEntity.fail("该手机号已注册");
    }
    if (passWord == null || passWord.length() < 6) {
        return ServerResponseEntity.fail("密码不能少于6位");
    }

    User user = new User();
    user.setPhone(phone);
    user.setNickName("用户" + RandomUtil.randomNumbers(4));
    user.setLoginPassword(passwordEncoder.encode(passWord));  // BCrypt 加密
    user.setRegisterType(1); // 手机号注册
    user.setStatus(1);
    userService.save(user);
    sendRegisterGiftCoupons(user.getId());

    return ServerResponseEntity.success(null);
}
```

### 4.4 后端管理员登录接口

**文件路径**：`kinetic-sports-admin/.../AuthController.java`（端口 8085）

```java
@PostMapping("/login")
public ServerResponseEntity<Map<String, Object>> login(@RequestBody Map<String, String> params) {
    String username = params.get("username");
    String password = params.get("password");

    // 1. 根据用户名查找管理员
    SysUser sysUser = sysUserService.getOne(
        new LambdaQueryWrapper<SysUser>().eq(SysUser::getUsername, username)
    );
    if (sysUser == null) {
        return ServerResponseEntity.fail("用户名或密码错误");
    }

    // 2. 密码校验：兼容 MD5、明文、BCrypt 三种方式
    String inputMd5 = md5(password);
    if (!sysUser.getPassword().equals(inputMd5)
        && !sysUser.getPassword().equals(password)
        && !matchesBcrypt(password, sysUser.getPassword())) {
        return ServerResponseEntity.fail("用户名或密码错误");
    }

    // 3. 账号状态检查
    if (sysUser.getStatus() != 1) {
        return ServerResponseEntity.fail("账号已被禁用");
    }

    // 4. Sa-Token 登录
    StpUtil.login(sysUser.getId());
    String token = StpUtil.getTokenValue();

    Map<String, Object> result = new HashMap<>();
    result.put("token", token);
    result.put("userId", sysUser.getId());
    result.put("nickName", sysUser.getNickName());
    return ServerResponseEntity.success(result);
}

// 获取当前登录用户信息
@GetMapping("/info")
public ServerResponseEntity<SysUser> info() {
    Long userId = StpUtil.getLoginIdAsLong();
    SysUser sysUser = sysUserService.getById(userId);
    sysUser.setPassword(null);  // 不返回密码字段
    return ServerResponseEntity.success(sysUser);
}

// 退出登录
@PostMapping("/logout")
public ServerResponseEntity<Void> logout() {
    StpUtil.logout();
    return ServerResponseEntity.success();
}
```

## 五、Sa-Token 鉴权机制

### 5.1 工作原理

Sa-Token 是一个轻量级 Java 权限认证框架，本项目用它来管理登录状态：

```
用户登录 → StpUtil.login(userId) → Sa-Token 生成 token → 存入 Redis
后续请求 → 携带 token（Header: Authorization）→ Sa-Token 拦截器校验 → 通过则放行
```

### 5.2 拦截器配置

**文件路径**：`kinetic-sports-security-common/.../SaTokenConfig.java`

```java
@Configuration
public class SaTokenConfig implements WebMvcConfigurer {

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(new SaInterceptor(handle -> StpUtil.checkLogin()))
                .addPathPatterns("/**")          // 拦截所有路径
                .excludePathPatterns(            // 白名单（不需要登录）
                        "/auth/login",
                        "/auth/wxLogin",
                        "/auth/register",
                        "/auth/smsLogin",
                        "/sms/send",
                        "/course/list",          // 课程列表公开
                        "/prod/list",            // 商品列表公开
                        "/recommend/home",       // 推荐接口公开
                        "/ai/chat",              // AI客服公开
                        "/banner/list",          // 轮播图公开
                        "/uploads/**"            // 上传文件公开
                        // ...
                );
    }
}
```

### 5.3 token 存储位置对比

| 端 | token 存储位置 | 读取方式 |
|----|----------------|----------|
| 小程序端 | `uni.setStorageSync('token', ...)` | `uni.getStorageSync('token')` |
| 管理后台 | `localStorage.setItem('token', ...)` | `localStorage.getItem('token')` |
| 后端 | Redis（Sa-Token 自动管理） | Sa-Token 框架自动读取 |

## 六、完整流程图

### 小程序用户登录

```
小程序登录页
  │
  ├── 密码登录 ──→ POST /auth/login ──→ 查 user 表 ──→ BCrypt 校验
  │                                                        │
  ├── 验证码登录 → POST /auth/smsLogin → 校验验证码 ──→ 查 user 表
  │                                                        │
  └── 微信登录 ──→ POST /auth/wxLogin → 换取 openId → 查 user 表
                                                           │
                                              ┌────────────┴────────────┐
                                              │ 用户存在                │ 用户不存在
                                              │ 直接登录                │ 自动注册
                                              └────────────┬────────────┘
                                                           │
                                                  StpUtil.login(userId)
                                                           │
                                                  生成 token（存 Redis）
                                                           │
                                                  返回 token + 用户信息
                                                           │
                                              小程序存入 localStorage
                                                           │
                                                  跳转回原页面
```

### 管理后台登录

```
管理后台登录页
  │
  └── 用户名+密码 → POST /api/auth/login（Vite代理到8085）
                        │
                        └── 查 sys_user 表 → 密码校验（MD5/BCrypt）
                                │
                                └── StpUtil.login(sysUserId)
                                        │
                                        └── 返回 token
                                                │
                                    localStorage.setItem('token')
                                                │
```
