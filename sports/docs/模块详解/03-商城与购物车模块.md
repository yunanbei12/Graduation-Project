# 商城与购物车模块详解

> 本模块覆盖商品浏览 → 加入购物车 → 购物车管理 → 确认订单 → 创建商品订单 → 支付 → 库存扣减的完整链路。
> 商品订单与课程订单共用同一套 OrderService，但在库存处理、地址快照、退款规则上有差异。

---

## 一、模块全链路概览

```
商城列表页（mall.vue）
  │ 点击商品卡片
  ▼
商品详情页（product-detail.vue）
  ├── 加入购物车 → POST /cart/add → 购物车页（cart.vue）→ 结算
  └── 立即购买  ─────────────────────────────────────────┐
                                                         ▼
                                              确认订单页（confirm-product.vue）
                                                         │
                                              POST /order/prod（创建商品订单）
                                                         │
                                              OrderServiceImpl.createProdOrder()
                                                         │
                                              校验商品/地址/手机号/优惠券
                                              写入 order + order_item 表
                                              保存地址快照（JSON）
                                              锁定优惠券
                                              清空购物车（购物车结算时）
                                              发布延迟关单消息（RabbitMQ）
                                                         │
                                              跳转订单详情页 → 用户点击支付
                                                         │
                                              PUT /order/pay/{id}
                                                         │
                                              分布式锁扣减 SKU 库存
```

```java
@GetMapping("/list")
public ServerResponseEntity<Page<Prod>> list(
        @RequestParam(required = false) Long categoryId,
        @RequestParam(required = false) String keyword) {
    LambdaQueryWrapper<Prod> wrapper = new LambdaQueryWrapper<>();
    wrapper.eq(Prod::getStatus, 1);  // 只查上架商品
    if (categoryId != null) wrapper.eq(Prod::getCategoryId, categoryId);
    if (keyword != null && !keyword.isBlank()) {
        // 关键词同时匹配商品名称和描述
        wrapper.and(w -> w.like(Prod::getName, keyword)
                .or().like(Prod::getDescription, keyword));
    }
    wrapper.orderByDesc(Prod::getSales);  // 按销量降序
    return ServerResponseEntity.success(prodService.page(page, wrapper));
}
```

## 四、购物车模块

### 4.1 加入购物车（后端）

**文件路径**：`ApiCartController.java`

```java
@PostMapping("/add")
public ServerResponseEntity<Void> add(@RequestBody Cart cart) {
    Long userId = StpUtil.getLoginIdAsLong();
    cart.setUserId(userId);

    // 检查是否已存在相同商品+SKU（合并数量，不重复创建）
    LambdaQueryWrapper<Cart> wrapper = new LambdaQueryWrapper<Cart>()
            .eq(Cart::getUserId, userId)
            .eq(Cart::getProdId, cart.getProdId());
    if (cart.getSkuId() != null) {
        wrapper.eq(Cart::getSkuId, cart.getSkuId());
    } else {
        wrapper.isNull(Cart::getSkuId);
    }
    Cart existCart = cartService.getOne(wrapper);
    if (existCart != null) {
        // 已存在：数量叠加
        existCart.setQty(existCart.getQty() + cart.getQty());
        cartService.updateById(existCart);
    } else {
        // 不存在：新增
        cartService.save(cart);
    }
    return ServerResponseEntity.success();
}
```

### 4.2 购物车列表（后端填充商品信息）

```java
@GetMapping("/list")
public ServerResponseEntity<List<Cart>> list() {
    Long userId = StpUtil.getLoginIdAsLong();
    List<Cart> cartList = cartService.list(...);

    // 填充商品名称、图片、SKU 价格（购物车表只存 prodId/skuId）
    cartList.forEach(cart -> {
        Prod prod = prodService.getById(cart.getProdId());
        if (prod != null) {
            cart.setProdName(prod.getName());
            cart.setProdPic(prod.getPic());
            if (cart.getSkuId() != null) {
                Sku sku = skuService.getById(cart.getSkuId());
                if (sku != null) {
                    cart.setSkuProperties(sku.getProperties());
                    cart.setSkuPrice(sku.getPrice());
                }
            } else {
                cart.setSkuPrice(prod.getPrice());  // 单规格用商品价格
            }
        }
    });
    return ServerResponseEntity.success(cartList);
}
```

> 购物车表只存 `prodId`、`skuId`、`qty`，商品名称和价格在查询时实时填充，保证展示的是最新价格。

```javascript
const totalPrice = computed(() => {
  const total = cartItems.value
    .filter(item => item.selected)
    .reduce((sum, item) => sum + item.price * item.qty, 0)
  return total.toFixed(2).replace(/\B(?=(\d{3})+(?!\d))/g, ',')  // 千分位格式化
})
```

## 六、后端创建商品订单

**文件路径**：`OrderServiceImpl.java` → `createProdOrder()`

### 6.1 完整逻辑

```java
@Transactional(rollbackFor = Exception.class)
public Order createProdOrder(Long userId, Order params) {

    // ① 用户校验
    User currentUser = requireUser(userId);
    ensurePhoneBound(currentUser);

    // ② 构建商品行（支持购物车结算和单品购买两种入口）
    List<ProductLine> lines = buildProductLines(userId, params);
    if (lines.isEmpty()) throw new IllegalArgumentException("购物车为空");

    // ③ 计算总金额
    BigDecimal totalAmount = BigDecimal.ZERO;
    for (ProductLine line : lines) {
        totalAmount = totalAmount.add(line.price.multiply(BigDecimal.valueOf(line.qty)));
    }

    // ④ 优惠券计算（orderType=2 表示商品订单）
    CouponPricing pricing = calculateCouponPricing(userId, params.getCouponId(), totalAmount, 2);

    // ⑤ 保存地址快照（JSON 序列化，防止用户后续修改地址影响历史订单）
    if (params.getAddressId() != null) {
        UserAddress address = userAddressService.getById(params.getAddressId());
        if (address != null && address.getUserId().equals(userId)) {
            order.setAddressSnapshot(objectMapper.writeValueAsString(address));
        }
    }

    // ⑥ 写入订单
    this.save(order);

    // ⑦ 写入订单项（每个商品一条记录）
    for (ProductLine line : lines) {
        OrderItem item = new OrderItem();
        item.setOrderId(order.getId());
        item.setItemType(2);
        item.setItemId(line.prod.getId());
        item.setItemName(line.prod.getName());
        item.setPrice(line.price);
        item.setQty(line.qty);
        item.setSkuId(line.sku == null ? null : line.sku.getId());
        item.setSkuProperties(line.sku == null ? null : line.sku.getProperties());
        orderItemService.save(item);
    }

    // ⑧ 锁定优惠券
    lockCouponForOrder(pricing.userCouponId(), userId, order.getId());

    // ⑨ 清空购物车（购物车结算时）
    if (params.getCartIds() != null && !params.getCartIds().isEmpty()) {
        cartService.removeByIds(params.getCartIds());
    }

    // ⑩ 事务提交后发布延迟关单消息
    txEventHelper.afterCommit(() ->
        orderEventPublisher.publishOrderCloseDelay(order.getId())
    );

    return order;
}
```

### 6.2 地址快照的意义

```java

// 存储内容示例：
```

## 七、商品订单支付：库存扣减

**文件路径**：`OrderServiceImpl.java` → `handleProdPay()`

```java
private void handleProdPay(Order order) {
    List<OrderItem> items = orderItemService.list(...);

    for (OrderItem item : items) {
        Prod prod = prodService.getById(item.getItemId());
        if (prod == null || prod.getStatus() != 1) {
            throw new IllegalArgumentException("商品不存在或已下架");
        }

        if (item.getSkuId() != null) {
            // 有 SKU 规格：加分布式锁扣减 SKU 库存
            RLock skuLock = redissonClient.getLock("order:sku:" + item.getSkuId());
            skuLock.lock();
            try {
                Sku sku = skuService.getById(item.getSkuId());
                if (sku.getStocks() < item.getQty()) {
                    throw new IllegalArgumentException("商品库存不足");
                }
                sku.setStocks(sku.getStocks() - item.getQty());  // 扣减库存
                skuService.updateById(sku);
            } finally {
                skuLock.unlock();
            }
        }

        // 更新商品销量
        prod.setSales(Math.max(0, prod.getSales()) + item.getQty());
        prodService.updateById(prod);
    }

    order.setStatus(ORDER_PAID);  // 商品订单支付后状态为"已支付"（等待发货）
}
```

> 注意：课程订单支付后状态变为 `ORDER_PROCESSING`（履约中），商品订单支付后状态为 `ORDER_PAID`（等待发货），两者不同。

## 八、退款时库存回补

退款审核通过后，商品库存会自动回补：

```java
// finalizeRefund() 中的商品退款处理
List<OrderItem> items = orderItemService.list(...);
for (OrderItem item : items) {
    // 回补销量
    prod.setSales(Math.max(0, prod.getSales()) - item.getQty());
    prodService.updateById(prod);

    if (item.getSkuId() != null) {
        // 加分布式锁回补 SKU 库存
        RLock skuLock = redissonClient.getLock("order:sku:" + item.getSkuId());
        skuLock.lock();
        try {
            Sku sku = skuService.getById(item.getSkuId());
            sku.setStocks(sku.getStocks() + item.getQty());  // 回补库存
            skuService.updateById(sku);
        } finally {
            skuLock.unlock();
        }
    }
}
```

## 十、完整流程图

### 购物车结算流程

```
商城列表页
  │ 点击商品卡片
  ▼
商品详情页（GET /prod/detail/{id}）
  │ 上报浏览行为（POST /behavior/view）
  │ 加载相似推荐（GET /recommend/prod/related/{id}）
  │ 点击"加入购物车"→ 弹出 SKU 弹窗 → 选择数量
  ▼
POST /cart/add { prodId, qty }
  ├── 检查是否已存在相同商品+SKU
  ├── 已存在：数量叠加
  └── 不存在：新增记录
  ▼
购物车页（GET /cart/list）
  │ 勾选商品 → 点击"结算"
  ▼
确认订单页（GET /cart/list + GET /coupon/usable + GET /address/default）
  │ 选择收货地址（可选）
  │ 选择优惠券（可选）
  │ 点击"提交订单"
  ▼
POST /order/prod { cartIds, addressId, couponId? }
  ├── 校验用户手机号
  ├── 构建商品行（从购物车读取）
  ├── 计算总金额
  ├── 计算优惠券金额
  ├── 保存地址快照（JSON）
  ├── 写入 order 表（status=1 待支付）
  ├── 写入 order_item 表（每个商品一条）
  ├── 锁定优惠券（status=3）
  ├── 清空已结算的购物车项
  └── 发布延迟关单消息（15分钟）
  ▼
跳转订单详情页（fromPay=1）
  │ 点击"立即支付"
  ▼
PUT /order/pay/{id}
  ├── 分布式锁（order:pay:{id}）
  ├── 更新订单状态（status=2 已支付）
  ├── 遍历订单项，逐个扣减 SKU 库存（分布式锁 order:sku:{id}）
  ├── 更新商品销量
  ├── 核销优惠券（status=1）
  └── 发布支付成功事件（异步发放满额优惠券）
  ▼
等待管理员发货（后台操作 → status=3 待处理）
  ▼
用户确认收货（PUT /order/confirm/{id} → status=4 已完成）
```

## 十一、课程订单 vs 商品订单对比

| 对比项 | 课程订单 | 商品订单 |
|--------|----------|----------|
| 订单类型 | orderType=1 | orderType=2 |
| 是否需要地址 | 否 | 是（保存地址快照） |
| 支付后状态 | ORDER_PROCESSING（履约中） | ORDER_PAID（等待发货） |
| 支付后操作 | 生成课包 或 占用团课名额 | 扣减 SKU 库存 |
| 退款时操作 | 课包失效 或 释放团课名额 | 回补 SKU 库存 |
| 优惠券适用范围 | scope=1（全场）或 scope=2（仅课程） | scope=1（全场）或 scope=3（仅商品） |
| 发货流程 | 无 | 后台发货 → 用户确认收货 |
| 售后时效 | 团课开课前 2 小时不可退 | 确认收货 7 天内可申请售后 |
