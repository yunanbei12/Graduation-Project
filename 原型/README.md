# 体育教培机构管理系统 - 小程序原型设计

## 📱 原型说明

这是一个高保真的微信小程序原型设计，包含了体育教培机构管理系统的所有核心页面。

## 🎨 设计特点

- **手机尺寸**: 严格按照 430x932 像素设计（iPhone 14 Pro尺寸）
- **主题色**: 橙色渐变主题（#ff6b35 到 #ff8c42）
- **布局方式**: 一排展示8个页面，自动换行
- **交互方式**: 点击任意页面可查看详情，支持滚动浏览
- **设计风格**: 简洁清爽，卡片式布局，高还原度

## 📄 包含页面（共16个）

### 第一行（8个页面）
1. **首页** - 推荐课程、热门拼团、推荐装备、精彩瞬间
2. **课程列表** - 课程筛选、分类浏览
3. **课程详情** - 课程信息、教练介绍、购买/拼团
4. **拼团中心** - 我发起的、可参与的、拼团记录
5. **我的课程** - 课包管理、上课记录、团课安排
6. **商品列表** - 运动装备展示、分类筛选
7. **商品详情** - 商品信息、规格参数、推荐课程
8. **购物车** - 商品管理、数量调整、结算

### 第二行（8个页面）
9. **精彩瞬间** - 小红书风格瀑布流动态展示
10. **动态详情** - 视频/图片详情、评论互动
11. **成长档案** - 学员信息、技能雷达图、体测数据、成就徽章、成长时间轴
12. **AI客服** - 智能问答、常见问题、对话历史
13. **个人中心** - 用户信息、功能入口、设置
14. **我的订单** - 订单列表、状态筛选、订单详情
15. **优惠券** - 优惠券列表、使用状态
16. **兴趣选座** - 新用户引导、偏好设置

## 🚀 使用方法

### 方法一：直接打开
1. 双击 `index.html` 文件
2. 使用浏览器打开（推荐Chrome、Safari、Edge）

### 方法二：本地服务器
```bash
# 进入原型目录
cd 原型

# 使用Python启动简单服务器
python3 -m http.server 8000

# 或使用Node.js
npx serve

# 然后在浏览器访问
# http://localhost:8000
```

## 💡 交互说明

- **点击页面**: 点击任意小程序页面可查看大图详情
- **关闭详情**: 点击右上角关闭按钮、按ESC键或点击背景区域
- **滚动浏览**: 详情页面支持滚动查看完整内容
- **响应式**: 自动适配不同屏幕尺寸

## 🎯 技术栈

- **HTML5**: 页面结构
- **Tailwind CSS**: 样式框架（CDN引入）
- **Font Awesome**: 图标库（CDN引入）
- **原生JavaScript**: 交互逻辑

## 📐 设计规范

### 颜色规范
- 主色调: `#ff6b35` - `#ff8c42` (橙色渐变)
- 成功色: `#4ade80` (绿色)
- 警告色: `#fbbf24` (黄色)
- 文字色: `#333` (深灰)
- 辅助色: `#999` (浅灰)

### 尺寸规范
- 手机宽度: 430px
- 手机高度: 932px
- 圆角: 12px (卡片), 20px (按钮), 50% (圆形)
- 间距: 10px, 15px, 20px

### 字体规范
- 标题: 16-24px, bold
- 正文: 13-14px, normal
- 辅助: 11-12px, normal
